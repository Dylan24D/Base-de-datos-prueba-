GitHub: El Historial

Supabase: El Cerebro

Backend as a Service (BaaS) con Auth simplificado, Edge Functions para lógica en la nube y seguridad a nivel de fila (RLS).

Key: BaaS

MySQL: La Biblioteca

Estructura relacional para la persistencia de datos. Garantiza que la información esté organizada lógicamente mediante un Schema definido.
